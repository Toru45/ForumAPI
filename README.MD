## Local Installation
1. Clone this repository
   ```
   git clone https://github.com/Toru45/ForumAPI.git
   ```
2. Change to directory project:
   ```
   cd ForumAPI
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Start The Server

```
npm run start
```
